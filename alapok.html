Változók és Adattípusok

A C# nyelvben a változó egy úgynevezett memóriahely, amelyet adat tárolására használnak. Az adattípus meghatározza, milyen adatokat lehet tárolni a változóban. Itt bemutatunk néhány gyakori adattípust, és hogy hogyan lehet változókat deklarálni és inicializálni.

  • Adattípusok:
  • int típusú változó: Az egész számokat reprezentálja.

    • Példa az int típusú változóra: int myNumber = 42; Ez a sor egy egész számot rendel a myNumber nevű változóhoz. Az "int" típus 32 bites egész számokat képes tárolni, és az értéke lehet pozitív vagy negatív.

  • float
  • A float típus 32 bites hosszúságú, és lebegőpontos számokat tárol a IEEE 754 szabvány szerint. Az előnye az, hogy kisebb memóriaterületet foglal, mint a double típus

    • Példa a float típusú változóra: float myFloat = 3.14f; A float típus lebegőpontos számok tárolására szolgál. A float típus különösen hasznos, amikor szükség van tizedespontos számokra.

  • double
  • A 'double' típusú változó általában tizedespontos lebegőpontos számokat reprezentál C#-ban.

    • példa a double típusú változóra:

    double 1 = 3.14159;

    double 2 = 50000.50;

    double 3 = 123456789.987654321;

  • char
  • A char típus különböző módon hasznos lehet, például szövegkezelés során, karakterek összehasonlítására vagy éppen karakterláncok összeállításakor.

    Példa a char típusú változóra:

    char grade = 'A';

    char digit = '5';

    char specialChar = '@';

  • string
  • A string típus hasznos szövegmanipulációs műveletek elvégzéséhez, például szöveg összefűzéséhez, kereséséhez, vágásához, helyettesítéséhez és még sok más művelethez.

    • Példa a string típusú változóra:

    string nev = "John";

    string udvozlet = "Hello, World!";

  • bool
  • A bool típusú változó C#-ban igaz/hamis értékek tárolására szolgál. A bool (boolean) típus két értéket vehet fel: true (igaz) vagy false (hamis). A bool típust a System.Boolean osztály reprezentálja.

    • Példa a bool típusú változóra:

    bool ehes = true;

    bool szomjas = false;

    A fenti példában az 'ehes' változó true értéket tárol, míg a 'szomjas' változó false értéket tárol.

Feltételes Utasítások

A feltételes utasítások lehetővé teszik a program számára, hogy döntéseket hozzon bizonyos feltételek alapján. A leggyakoribb feltételes utasítások a C# nyelvben az if utasítások és a switch utasítások.

  • if utasítás: if (x > 0) { Console.WriteLine("x pozitív."); }

  • switch utasítás: switch (x) { case 1: Console.WriteLine("x egy."); break; }

Ciklusok

Ciklusok lehetővé teszik a program számára, hogy többször ismételje meg egy kódblokk végrehajtását. A leggyakoribb ciklusok a C# nyelvben a for ciklusok és a while ciklusok.

  • for ciklus: for (int i = 0; i < 10; i++) { Console.WriteLine(i); }
  • while ciklus: int i = 0; while (i < 10) { Console.WriteLine(i); i++; }

Függvények

A függvények újrafelhasználható kódblokkok, amelyek egy konkrét feladatot látnak el. A függvények inputokat vehetnek át, és kimeneteket adhatnak vissza. Itt bemutatjuk, hogyan lehet deklarálni és hívni függvényeket C# nyelven.

  • Függvények deklarálása és hívása: int Összeadás(int a, int b) { return a + b; }
